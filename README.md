# 📖 Overview

ResuMetrics is a modern AI-powered Resume Intelligence Platform designed to help students, developers, and professionals understand how recruiters and Applicant Tracking Systems (ATS) evaluate resumes.

Instead of simply generating an ATS score, ResuMetrics performs comprehensive resume intelligence by analyzing structure, technical skills, credibility, projects, experience, keyword optimization, semantic quality, and hiring readiness.

The platform provides actionable insights that help candidates improve their resumes before applying for jobs.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/alimanpur/ResumeMetrics_Day-01.git
```

```bash
cd ResumeMetrics_Day-01
```

---

## Backend

```bash
cd backend
npm install
```

Create `.env`

```env
DATABASE_URL=

JWT_SECRET=

JWT_REFRESH_SECRET=

CLOUDINARY_NAME=

CLOUDINARY_KEY=

CLOUDINARY_SECRET=

AI_PROVIDER=mock
```

Run

```bash
npm run dev
```

## Frontend

```bash
cd frontend
```
